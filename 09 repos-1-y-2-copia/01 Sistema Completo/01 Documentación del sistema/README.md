## Principios del sistema

- **Web first, GBP despuГ©s.** El GBP se crea en el Paso 14, despuГ©s de publicar la web base.
- **Main City Гәnica** crea la arquitectura. No se multiplica por zonas.
- **Local Coverage Areas no generan URLs por defecto.** Refuerzan contenido y schema.
- **Approved Expansion Areas** generan URLs solo con aprobaciГіn explГӯcita.
- **28 pГЎginas SEO base:** `1 + S + 1 + S + A + GГ—S` (con S=5 servicios, A=1 categorГӯa adicional, G=3 GeoArticles).
- **GBP Status = Not Created hasta el Paso 14.** Antes: categorГӯas "Planned", sin GBP URL en schema, sin sameAs, sin reseГұas.

Las 28 reglas estructurales completas: ver `reglas-del-sistema.md`.

## Trazabilidad de decisiones

Los archivos `XXb-decisiones-tomadas-y-fuentes.md` usan **7 columnas**:

```text

```

- **Fuentes permitidas:** GMB Crush, Input humano, DecisiГіn de diseГұo, Competidores, Datos de bГәsqueda, IA sin respaldo.

- **Dato en cascada:** `no` o `вҶҗ X.YY`.
