# Carpeta de trabajo

Esta carpeta contiene materiales auxiliares que se usan durante el proceso de auditoría y trazabilidad del sistema GMB Crush.

Su propósito es mantener borradores, versiones intermedias y documentos de análisis separados de los archivos originales del sistema, que están en las carpetas de Bloque y no deben modificarse.

Los archivos `.md` originales de cada paso (los `XXa-ejecucion-*.md`) son la fuente de verdad del sistema. Los documentos de esta carpeta de trabajo se generan a partir de ellos, pero nunca los sustituyen ni los alteran.
