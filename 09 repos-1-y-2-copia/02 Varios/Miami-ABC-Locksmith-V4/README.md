# Sistema GMB Crush para webs locales — Versión corregida V4

Este directorio contiene los 13 documentos del sistema GMB Crush para construir una web local desde cero.

Esta versión corrige el error detectado en la V3: los bloques repetidos de “Regla operativa N” fueron reemplazados por reglas reales, únicas y específicas para cada paso.

## Lógica base del sistema

```text
Main City = crea la arquitectura base.
Local Coverage Areas = enriquecen contenido, FAQs, ejemplos locales y schema areaServed.
Approved Expansion Areas = generan URLs propias solo cuando se aprueban.
```

## Validación aplicada

- Se mantiene la estructura fija: cabecera, índice corto, objetivo, campos a rellenar, ejemplo rellenado, cuerpo operativo, checklist y outputs.
- Se mantiene el caso ABC Locksmith como ejemplo operativo.
- Se eliminan reglas operativas duplicadas.
- Cada regla operativa tiene nombre semántico y contenido específico.
- Las Local Coverage Areas no generan URLs por defecto.
- Approved Expansion Areas quedan como módulo opcional.
