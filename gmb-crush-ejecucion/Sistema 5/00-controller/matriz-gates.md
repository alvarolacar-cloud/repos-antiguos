## Gate 0 — Teoría leída

| Check | Criterio | Resultado |
|---|---|---|
| README teoría leído | La IA declara haber leído README y convenciones | ⬜ |
| Fuentes leídas | La IA conoce catálogo de fuentes | ⬜ |
| Flujo leído | La IA entiende Bloques 1–7 | ⬜ |
| No modificar teoría | La IA entiende que no cambia documentos canónicos | ⬜ |
