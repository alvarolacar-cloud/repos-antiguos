# Check 03 — Referencias canónicas

## Contrato

`Funcionamiento` solo puede contener:

```text
Paso-XX §Y
Paso-XX §Y.Z
Paso-XX §Y; Paso-ZZ §W
```

No puede contener:

```text
texto libre
framework externo
herramientas
criterio N si no existe heading real
rangos no anclables
```

---

## Checklist

| Check | Estado |
|---|---|
| 0 referencias `§X` | ⬜ |
| 0 referencias sin `Paso-XX` | ⬜ |
| 0 strings libres en Funcionamiento | ⬜ |
| 0 rangos tipo `Paso-01–Paso-16` | ⬜ |
| 0 referencias a secciones inexistentes | ⬜ |
