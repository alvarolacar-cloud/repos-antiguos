# Prompt 01 — Validar preflight

```text
Actúa como validador de preflight.

Lee el preflight del cliente.

Clasifica cada campo como:
✅ Validado
⚠ Inferido
⚠ Placeholder
⛔ Bloqueado
🔁 Revisión

Indica:
- qué falta
- qué puede inferirse
- qué bloquea web
- qué bloquea GBP
- si se puede ejecutar Bloques 1–5

No inventes datos.
No crees GBP.

Devuelve:
1. Tabla de validación
2. Faltantes
3. Decisión de gate
```
